INSTRUCOES PARA A COMPILACAO (LINUX):

No diretorio raiz do projeto, basta entrar com o comando "make". O executavel
sera criado e colocado no diretorio "./bin".


INSTRUCOES PARA A EXECUCAO (LINUX):

Basta executar o comando "./bin/gerenciador_arq_led.out <PARAMETROS>"

Em <PARAMETROS> eh possivel inserir os seguintes modos:

"-r <CAMINHO>": Especifica o caminho desejado para o arquivo de registro a ser
editado. Caso nao seja especificado, o padrao usado na execucao eh "./res/rezistro.rez".


INSTRUCOES PARA A COMPILACAO (WINDOWS):

Usando o MinGW, as instrucoes sao semelhantes aas usadas para compilar no Linux. Para compilar atraves de uma IDE, eh necessario se criar um novo projeto com os arquivos main.c, arq_reg.c, arq_reg.h, arvore_b.c e arvore_b.h. Nenhuma biblioteca adicional eh necessaria.
